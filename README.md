# Story Fellow

Story Fellow turns written content into high-quality narrated audio.
It is designed for creators, teams, and products that want fast text-to-audio publishing.

## Product Summary
- Convert text files and web articles into narrated audio
- Generate downloadable audio assets asynchronously
- Track job progress from upload to final output
- Support creator workflows with reusable voice settings

## Why This Product
Publishing audio versions of articles, stories, and updates is usually manual and time-consuming. Story Fellow automates that workflow so teams can produce listening-ready content with predictable delivery.

## Core Capabilities
- Text upload to narration pipeline
- URL-to-podcast job creation for online articles
- Async speech generation with job status tracking
- Output delivery through secure access URLs
- Optional distribution path for public playback

## Typical User Flow
1. User creates a narration job and uploads content.
2. Story Fellow starts asynchronous synthesis.
3. User tracks job state in real time.
4. Completed audio and metadata become available for playback or download.

## Who This Is For
- Content teams and bloggers
- Education and media products
- Developer teams building audio features
